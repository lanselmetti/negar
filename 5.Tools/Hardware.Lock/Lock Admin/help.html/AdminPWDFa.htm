رمز عبور Admin قفل که حداکثر می تواند 16 کاراکتر باشد برای برنامه ریزی قفل استفاده می شود و جایگاه استفاده آن فقط در نرم افزار برنامه ریزی قفل می باشد .
این رمز برای تغییر در پارامترهای قفل بعدا مورد استفاده قرار می گیرد و وجود آن به جلوگیری از دستکاری در پارامترهای قفل کمک می کند. همچنین دانستن آن برای Reset کردن یا خارج کردن قفل از حالت غير فعال (disable) لازم است.
