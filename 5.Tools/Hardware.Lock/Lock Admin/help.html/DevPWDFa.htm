رمز عبور برنامه نویس همانطور که از اسمش پیداست جهت استفاده از برخی سرویسهای قفل در برنامه نویسی مورد استفاده قرار می گیرد.
این رمز برای سرویسهای زیر استفاده می شود:

  • رمزنگاری AES اصلی (با استفاده از کلید اصلی) در GetEncryption

  • خواندن قسمتی از حافظه قفل با استفاده از سرویسهای GetByte، GetChar، GetBlockstr و GetBlockHexstr

  • نوشتن در قسمتی از حافظه قفل با استفاده از SetByte، SetChar، SetBlockstr و SetBlockHexStr
