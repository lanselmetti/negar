راهنمای كاربری سیستم مدیریت تصویربرداری سپهر

  معرفی فرم اصلی
 

تمامی برنامه های شركت رایان پرتو نگار با استفاده از سیستم مدیریت نرم افزار های كلینیكی آرش فرا خوانی می شوند. در این فرم می توانید با انتخاب تنظیمات گذشته خود و یا ایجاد تنظیمات جدید به هر یك از زیر سیستم ها متصل شوید.



درقسمت نام، می توانید برای تنظیمات  پیش رو نامی نمایشی انتخاب كنید. این نام در لیست اتصالات ذخیره شده نمایش داده خواهد شد.در صورتی كه دستگاه پیش رو سرور مركز می باشد، می توانید گزینه اتصال به سرور محلی را انتخاب نمایید. در غیر این صورت باید نام سرور یا IP آن را وارد كنید. نام سرور پیش فرض شركت پرتونگار Aftab01 و IP پیش فرض 172.16.16.1می باشد. برای اطلاع از نحوه یافتن نام سرور یا IP آن به قسمت پیشرفته  مراجعه نماید. نام نمونه بانك اطلاعاتی را در هنگام راه اندازی مركز از كارشناسان شركت پرتونگار در یافت كرده اید كه باید در این قسمت وارد شود.



با كلیك بر روی قسمت نرم افزار های  نصب شده ، لیستی از نرم افزار هایی كه قفل سخت افزاری شما مجوز آنها را دارا می باشد را مشاهده خواهید نمود. با انتخاب یك برنامه برای اولین بار با پیغام زیر روبه رو خواهید شد.



با كلیك بر روی دكمه بلی باید آدرس فایل اجرایی برنامه را مشخص نمایید.
با كلیك بر روی قسمت نام كاربری لیستی از نام های كاربری كه در سیستم ذخیره شده اند را مشاهده خواهید نمود. با انتخاب گزینه ذخیره نام كاربری، هنگامی كه از اتصالات ذخیره شده استفاده می كنید نیازی به انتخاب مجدد نام كاربری نخواهید داشت.

 پیشرفته
 برای یافتن نام سرور بر روی این دستگاه در RUN عبارت CMD را تایپ كنید. در صفحه پیش رو عبارت  HostName را تایپ نمایید.نام دستگاه نمایش داده خواهد شد.
برای یافتن IP سرور بر روی این دستگاه در RUN عبارت CMD را تایپ كنید. در صفحه پیش رو عبارت IPConfig را تایپ نمایید.
