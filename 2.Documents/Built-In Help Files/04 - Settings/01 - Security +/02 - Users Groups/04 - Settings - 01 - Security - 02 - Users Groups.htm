راهنمای كاربری سیستم مدیریت تصویربرداری سپهر

  معرفی فرم مدیریت گروه های كاربری سیستم
از این فرم برای افزودن و یا حذف گروه های كاربری سیستم استفاده نمایید. توجه داشته باشید كه می توانید گروه های كاربری بدون استفاده را غیر فعال نمایید.
برای حذف گروه كاربری خاص بر روی آن كلیك سمت راست نمایید. با فرم زیر رو به رو خواهید شد.
توجه داشته باشید كه در صورت حذف گروه كاربری تمامی كاربران عضو آن و دسترسی های تعریف شده برای آن نیز حذف خواهند شد.
